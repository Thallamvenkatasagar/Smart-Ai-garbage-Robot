 **Project Title:Smart AI-Based Garbage Collection and Segregation Robot**
**Project Overview**:This project aims to develop an autonomous AI-based garbage collection robot for smart cities. The robot uses a camera and AI model to classify waste into dry, wet, and medical categories.
**Features:**
Autonomous street navigation
AI-based waste classification
Three waste compartments
Fill-level detection sensors
GPS-based route optimization
Solar + Plug-in hybrid power
Smart city integration
**Technologies Used:**
Raspberry Pi
Camera Module
Ultrasonic Sensors
GPS Module
Solar Panel
AI Model (Image Classification)
**Objective**:To improve waste management efficiency, reduce manual labor, and promote smart city development.
